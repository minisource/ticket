# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within any Minisource project, please report it responsibly.

**Do NOT open a public issue.**

Instead, please email us at: **security@minisource.ir**

We will acknowledge your email within **48 hours** and provide a detailed response within **5 business days** indicating the next steps.

## Disclosure Policy

- We will work with you to understand and resolve the issue quickly.
- We will credit you (if desired) once the vulnerability is fixed.
- We ask that you do not publicly disclose the issue until we have had a chance to address it.

## Scope

This policy applies to all repositories under the **minisource** GitHub organization.

## Best Practices

- Keep dependencies up to date.
- Use Dependabot alerts and security advisories.
- Report any suspicious activity to the security team.
